# Restro Landing Page

A simple and responsive **Restaurant Landing Page** built using **HTML and CSS**.
This project showcases a modern UI layout for a restaurant website, including sections like a hero banner, food showcase, and contact information.

## Live Demo

https://tanushree2154.github.io/Restro-landing-page/

## Tech Stack

* **HTML5** – for structuring the webpage
* **CSS3** – for styling and layout
* **GitHub Pages** – for deploying the website

## Features

* Responsive restaurant landing page design
* Attractive hero section
* Food/menu showcase section
* Clean and modern UI layout
* Simple navigation bar
* Footer section
* Hosted live using GitHub Pages
